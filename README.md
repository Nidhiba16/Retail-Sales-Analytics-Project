# Retail Sales Analytics Project

## Project Overview
This project analyzes retail sales data to generate business insights such as monthly performance, customer trends, and category-wise revenue.

## Tools Used
- SQL
- Python (Pandas)
- Power BI

## Key Objectives
- Analyze monthly sales trends
- Identify top customers
- Track repeat customers
- Measure category performance

## Project Structure
- data/ → Raw and cleaned datasets
- notebooks/ → Python scripts for data cleaning
- sql/ → SQL analysis queries
- dashboard/ → Power BI dashboard file
